<p>Developers innovate at a fast pace. To help developers build voice and text messaging communication into applications, we’re unveiling the public beta of <a href="https://www.plivo.com/phlo/">PHLO (Plivo High-Level Objects)</a>, a new visual way of integrating communications that empowers developers to design collaboratively, build visually, and deploy instantly.</p><p>PHLO offers:</p><ul><li>Prebuilt use case templates with option to add custom code</li><li>100% functional parity with Plivo APIs and XML</li><li>Serverless architecture, hosted on Plivo infrastructure</li></ul><p>And to top it off, PHLO is free for all Plivo customers.</p><h3>Why it matters</h3><p>Although communication APIs have simplified the process, building communication capabilities into business applications is still a complex task for developers. We saw a need to simplify it.</p><figure style="padding-bottom:33.75%" data-rt-max-width="" data-rt-max-height="33.75%" data-rt-dimensions="0:0" data-page-url=""><div><iframe src="https://www.youtube.com/embed/jZmtRTe_vQo?rel=0"></iframe></div></figure><p>PHLO takes a visual drag-and-drop approach to building communication applications. It lets you build applications by connecting together graphical components in much the same way you’d piece together Legos.</p><figure style="max-width:1430px" data-rt-max-width="1430px"><div><img src="/images/blog/6582f55f7ee8de42c50b726a_Legos.png" alt="Legos" width="auto" height="auto" loading="auto"></div></figure><p>Building with PHLO is visual and intuitive. PHLO eliminates the need to keep referring back to documentation, since text-based help is available within the main workspace.</p><figure style="max-width:1281px" data-rt-max-width="1281px"><div><img src="/images/blog/6582f55fb82c726e424b70ac_workspace.gif" alt="workspace" width="auto" height="auto" loading="auto"></div></figure><p>With PHLO, tasks are components: phone menus, prompts, SMS menus, conference bridges, call forwarding, SMS notifications. These components, prepackaged as templates, drastically reduce the code required to create custom business logic. PHLO works seamlessly with all programming languages and is prepackaged within Plivo Server SDKs. You can still add your own custom code and create a unique communication application that’s relevant for your business — just now you can do it with 80% less coding.</p><p>Unlike visual application builders that only automate simple workflows for a limited set of use cases, PHLO provides 100% functional parity with Plivo API and XML set, enabling use cases from simple ones like order notifications to complex ones like a fully featured call center. PHLO also simplifies the integration and deployment pipeline so that you can ship more applications faster.</p><p>PHLO runs on Plivo’s cloud infrastructure. Because it’s hosted, you can focus on your business use case rather than worrying about all the stuff under the hood, such as setting up a server and writing code for routing and API calls.</p><h3>Getting started with PHLO</h3><p>The best way to get started with PHLO is to <a href="https://console.plivo.com/accounts/register/">sign up</a> for Plivo (if you’re not already using it) and try it. PHLO has everything you need to create an application in minutes. We’ve also put together a collection of use case video tutorials to get you started quickly.</p><ul><li><a><strong>Voice Notification</strong></a></li><li><a><strong>Voice Alert</strong></a></li><li><a><strong>SMS Alert</strong></a></li><li><a><strong>SMS Survey</strong></a></li><li><a><strong>Conference with PIN</strong></a></li><li><a><strong>PINless Conference</strong></a></li><li><a><strong>Call Forwarding</strong></a></li><li><a><strong>Phone System IVR</strong></a></li></ul><p>We also have plenty of documentation, in the form of a <a href="https://www.plivo.com/docs/phlo/">PHLO getting started guide</a> and <a href="https://www.plivo.com/docs/phlo/use-case-guides/call-forwarding/">detailed use case guides</a>.</p><h3>One last thing — the pricing :)</h3><p>At Plivo, our aim has always been to simplify integrating communications for developers. Sometimes, all we need to do is to unblock things for developers so that they can change the world. That’s why, in addition to offering 100% Plivo API support out of the box, we’re also making PHLO free. This is not just a commercial decision. This is our stake in the ground — we believe this is how the communication capabilities of the future will be built.</p><p><a href="https://console.plivo.com/accounts/register/">Sign up</a> and start building with for PHLO today.</p><p>Have questions? <a href="https://www.plivo.com/contact/sales/">Get in touch with an expert and request a live demo</a>.</p><p>Check out our launch announcement at <a href="https://www.producthunt.com/posts/phlo-by-plivo">Product Hunt</a> and leave your feedback.</p>
