<p>We’re so excited to have members of <a href="https://twitter.com/pyladiessf">PyLadies SF</a> join us this weekend to develop their Python skills. <a href="https://pyladies.com/">PyLadies</a> is an international mentorship group with a focus on helping more women become active participants and leaders in the Python open source community. Their mission is to promote, educate, and advance a diverse Python community through outreach, education, conferences, events, and social gatherings.</p><p>During the <a href="https://www.meetup.com/PyLadiesSF/events/111994152/">two-day workshop</a>, participants:</p><ol><li>Learn Python basics, including types, containers, functions, iterators and generators.</li><li>Learn how to create a web application in Python using Flask.</li><li>Use the Plivo package to learn how to make outbound telephone calls.</li><li>Create an interactive voice response application.</li><li>Learn how to create a conferencing application to connect multiple users using <a href="https://www.plivo.com/docs/sdk/server/python-sdk/">Python</a> and Plivo.</li></ol><p></p>
