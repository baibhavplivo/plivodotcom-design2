<p>We’re excited to announce the launch of the <a href="https://docs.microsoft.com/en-gb/connectors/plivo/">Plivo connector</a> for <a href="https://powerautomate.microsoft.com/en-US/">Microsoft Flow</a>. It’s a premium connector that you can use in your flows to send text messages and place calls in tandem with more than 200 other connectors, including ones for Microsoft Office and Azure. We’ve written a <a href="https://www.plivo.com/docs/integrations/microsoft-power-automate/">getting started guide</a> that shows you everything you need to know to start using the Plivo connector.</p><p></p>
