<p>SIP trunking lets you add phone lines that bypass the public switched telephone network (PSTN) and use VoIP instead. SIP stands for Session Initiation Protocol, while trunking refers to an aggregation of multiple phone lines. Many businesses are moving to SIP trunking and away from traditional telephony to benefit from advantages such as quick provisioning, low costs, and no equipment to procure or maintain. <a href="https://www.plivo.com/docs/sip-trunking/" target="_blank">SIP Trunking</a> is Plivo’s SIP trunking service.</p><p>You can create and manage outbound and inbound trunks from the Plivo console. The SIP Trunking menu is the fifth icon on the console menu. When you click it, you’ll see an overview screen that gives you quick instructions on creating trunks.</p><figure style="max-width:1968px" data-rt-max-width="1968px"><div><img src="/images/blog/657d4736e263ba3262e3f614_ztour01.png" loading="lazy" width="auto" height="auto" alt=""></div></figure><p>Following the Overview submenu are choices for Outbound Trunks and Inbound Trunks. Under each, you can see a list of existing trunks and, as on the overview screen, there’s a button to create new ones.</p><p>Clicking on a trunk in the list brings up details about it. Here, for example, is what you might see for an outbound trunk.</p><figure style="max-width:1964px" data-rt-max-width="1964px"><div><img src="/images/blog/657d4759d18a2454f1ff7fc2_ztour02.png" loading="lazy" width="auto" height="auto" alt=""></div></figure><p>You can modify trunk details and settings from this page, or delete a trunk you no longer use. The settings are covered in detail in our <a href="https://www.plivo.com/docs/sip-trunking#outbound-trunks-termination" target="_blank">documentation</a>.</p><p>You can get details about inbound trunks the same way by clicking on an item in the Inbound Trunks list.</p><figure style="max-width:1958px" data-rt-max-width="1958px"><div><img src="/images/blog/657d477e45a3814e215428b2_ztour03.png" loading="lazy" width="auto" height="auto" alt=""></div></figure><p>If you want to set up a new trunk, our documentation guides you through the process and explains all of the fields and settings.</p><p>Moving down, the Other Settings menu brings up a screen that lets you set up <a href="https://www.plivo.com/docs/sip-trunking/concepts/geo-permissions" target="_blank">geo permissions</a>, which you can enable to help curb fraud.</p><figure style="max-width:1044px" data-rt-max-width="1044px"><div><img src="/images/blog/657d47abd61bd50a76d92b0c_ztour04.png" loading="lazy" width="auto" height="auto" alt=""></div></figure><p>Another Other Settings menu page lets you consent to Plivo storing and accessing logs and media packet capture files, which we can use to debug call quality and DTMF issues.</p><p>The penultimate menu choice, Logs, displays a log of outbound and inbound SIP calls. Clicking on a log entry displays a Message Details popup that gives you information about the call.</p><figure style="max-width:1066px" data-rt-max-width="1066px"><div><img src="/images/blog/657d47d774a255e1b4d48aad_ztour05.png" loading="lazy" width="auto" height="auto" alt=""></div></figure><p>Finally, the pricing menu page gives you the costs for calls and phone number rental in every country in which Plivo supports SIP trunking.</p><h2>Step right up for another tour</h2><p>That’s a whirlwind tour through the SIP Trunking features on the Plivo console. We’ve got similar looks for the <a href="http://plivo-webflow.webflow.io/blog/console-tour-voice" target="_blank">Voice</a>, <a href="http://plivo-webflow.webflow.io/blog/console-tour-sms" target="_blank">Messaging</a>, <a href="http://plivo-webflow.webflow.io/blog/number-lookup-intro" target="_blank">Lookup</a>, and <a href="http://plivo-webflow.webflow.io/blog/console-tour-numbers" target="_blank">Phone Numbers</a> menus.</p>
