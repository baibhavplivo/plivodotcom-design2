<p>In this age of agile development and rapid prototyping, no one wants to spend weeks building communication applications. In our continuous endeavor to simplify business communications, we created a whole new way of building communication apps — <a href="https://www.plivo.com/phlo/">Plivo High-Level Objects</a>. PHLO drastically reduces coding time and cuts integration steps, thereby helping developers design collaboratively, build visually, and deploy quickly — and now it’s available in general release.</p><p>We’ve added several features to PHLO to enhance how workflows can interact with your data.</p><ul><li>The <a href="https://www.plivo.com/docs/phlo/components/#http-request">HTTP request component</a> offers more authentication options and support for static and dynamic URLs make interaction with your APIs or third-party resources easy.</li><li>With the <a href="https://www.plivo.com/docs/phlo/components/#http-request">HTTP request component</a>, you can parse multilevel JSON files to interact with larger and more complex data. &nbsp;</li><li>The <a href="https://www.plivo.com/docs/phlo/components/#branch">branch component</a> now automatically typecasts variables when comparing to ensure variables are correctly processed and help you build better logic for your workflows.</li><li>You can now assign PHLOs to subaccounts and have one PHLO trigger another, which helps with account management and application logic. &nbsp;</li></ul><p><strong>Fortified for scale and uptime</strong></p><p>PHLO eases the way for you to deploy and maintain communication applications. You don’t host your apps, we do. This convenience for you comes with added responsibility for us. We’ve engineered for reliability, quality, and 99.95% uptime to make sure your applications scale with ease. &nbsp;</p><p><strong>Server-side SDK and debugging</strong></p><p>We’ve added more ways to control your PHLO with API requests. You can now trigger your PHLO application via an API request using our <a href="https://www.plivo.com/docs/phlo/#triggering-a-phlo-using-an-api-request">SDKs</a> in all supported languages. The <a href="https://www.plivo.com/docs/phlo/components/#multi-party-call">multiparty call component</a> has multiple triggers to asynchronously modify the states of the members on the call. You can now add members to a call and make state changes like mute, unmute, transfer, and disconnect a member using a Plivo SDK.</p><p>In addition, we’ve added more visibility on application logs in your account to provide better debugging and error correction.</p><p><strong>A better way to build communication apps</strong></p><p>With PHLO, every action is a component: phone menus, prompts, SMS menus, conference bridges, call forwarding, SMS notifications. These components, packaged as templates, drastically reduce the number of lines of code required to create custom business logic. You can add your own custom code and create unique communication applications for your business with 80% less code.</p><p>Unlike visual application builders that only automate simple workflows for a limited set of use cases, PHLO provides 100% functional parity with Plivo APIs and XML, enabling use cases from simple ones like order notifications to complex ones like a full-featured call center. PHLO also simplifies the integration and deployment pipeline so that you can ship more code faster.</p><p>PHLO runs on Plivo’s cloud infrastructure. This lets you focus on your actual business use case, rather than worrying what’s under the hood.</p><p><strong>Getting started with PHLO</strong></p><p>The best way to get started is to <a href="https://console.plivo.com/accounts/register/">sign up</a> and create your first PHLO. PHLO has everything you need to create your app in minutes. We’ve also put together a collection of use case video tutorials to get you started quickly.</p><ul><li><a href="https://www.plivo.com/docs/voice/use-cases/voice-notification/node/">Voice notification</a></li><li><a href="https://www.plivo.com/docs/voice/use-cases/voice-alerts/node/">Voice alert</a></li><li><a href="https://www.plivo.com/docs/sms/use-cases/sms-alert/node/">SMS alert</a></li><li><a href="https://www.plivo.com/docs/sms/use-cases/sms-survey/node/">SMS survey</a></li><li><a href="https://www.plivo.com/docs/voice/use-cases/conference-with-pin/node/">Conference with PIN</a></li><li><a href="https://www.plivo.com/docs/voice/use-cases/call-conference/node/">PINless conference</a></li><li><a href="https://www.plivo.com/docs/voice/use-cases/call-forwarding/node/">Call forwarding</a></li><li><a href="https://www.plivo.com/docs/voice/use-cases/ivr/node/">Phone system IVR</a></li></ul><p><strong>One last thing — the price</strong></p><p>In addition to bringing in 100% Plivo API support out of the box, we’re also making it FREE to use PHLO.</p><p><a href="https://console.plivo.com/accounts/register/">Sign up</a> and start building with PHLO today.
