<p>You can now protect your Plivo user credentials using two-factor authentication. 2FA provides an additional level of security by asking for a unique verification code that Plivo sends to your mobile device when you log in into your account. Using 2FA to secure your account is optional, but we recommend doing so to better protecting your customer and transaction information.</p><p>We support text messaging (SMS) as the primary method for 2FA, and plan to add support for voice calls and mobile app-based one-time passwords (OTP) going forward.</p><h2>How 2FA works on the Plivo console</h2><p>2FA applies every time a user logs in to their account. After the user enters a username and password, they are requested to enter a verification code if 2FA is enabled for the user. A six-digit verification code is sent to the user’s verified phone number via an SMS message, and this verification code should be entered in the appropriate console screen.</p><figure style="max-width:1395px" data-rt-max-width="1395px"><div><img src="https://uploads-ssl.webflow.com/656ac26f3a3f6e1cc08ba7a5/6583c9f994892bcf3352740d_Enabling-Two-Factor.png" alt="Enabling Two Factor" width="auto" height="auto" loading="auto"></div></figure><h2>Enabling 2FA for your account</h2><p>You can enable two-factor authentication from the <a href="https://console.plivo.com/account/password/change/">Security</a> page of the console, under Settings. The account must have a verified phone number on the user’s profile to use 2FA.</p><figure style="max-width:1393px" data-rt-max-width="1393px"><div><img src="https://uploads-ssl.webflow.com/656ac26f3a3f6e1cc08ba7a5/6583c9f922edf196910d09a3_verification-process.png" alt="Verification Process" width="auto" height="auto" loading="auto"></div></figure><p>A verification process is triggered if a verified phone number is not available. To verify the phone number, we send an OTP to your phone number to be entered in the appropriate screen.</p><figure style="max-width:1392px" data-rt-max-width="1392px"><div><img src="https://uploads-ssl.webflow.com/656ac26f3a3f6e1cc08ba7a5/6583c9f9013c5dbea9e1b0c7_2FA.png" alt="2FA" width="auto" height="auto" loading="auto"></div></figure><p>Enable 2FA to keep your accounts secure.</p>
