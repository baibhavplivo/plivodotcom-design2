<p><a href="https://www.plivo.com/">Plivo</a> has a lot of great capabilities built into its cloud communications platform. If you’re a Python developer, this post is for you: Learn how easy it is to send and receive SMS text messages and make and receive voice calls with Python, the Plivo Python SDK, and Plivo APIs.</p><h2>How to send and receive SMS messages</h2><p>If you’re interested in messaging applications, Plivo’s cloud communications platform makes things easy for you. <a href="https://www.plivo.com/blog/send-sms-in-python/" target="_blank">Sending an SMS message</a> requires just eight lines of code; receiving messages takes just 16, or you can <a href="https://www.plivo.com/blog/receive-respond-sms-node-expressjs-plivo/" target="_blank">create an application with Express to receive and respond to SMS messages</a>.</p><h2>How to send and receive MMS messages</h2><p><a href="https://www.plivo.com/blog/send-mms-in-python/" target="_blank">Sending MMS messages</a> is equally simple, and we’ve written a <a href="https://www.plivo.com/blog/how-to-receive-and-respond-incoming-mms-messages-in-python/" target="_blank">guide to receiving and responding to MMS messages with Python</a>.</p><h2>How to get set up and make and receive phone calls</h2><p>Start by reading <a href="https://www.plivo.com/blog/make-phone-calls-in-python/" target="_blank">how to make a phone call from a Python application using Plivo’s Voice API</a>. That post covers how to install Plivo’s Python SDK, which makes it easier to build applications that use Plivo. The SDK provides objects, functions, and methods to handle common tasks associated with building voice and messaging applications.</p><p>Once you have that in place, you can use <a href="https://www.plivo.com/phlo/" target="_blank">PHLO</a>, Plivo’s visual workflow design studio, to set up the workflow of the call, then trigger the PHLO with a few lines of Python code.</p><p>That blog post also talks about how to receive phone calls using PHLO, but if you’re not a fan of visual design tools that make coding tasks easier, you can <a href="https://www.plivo.com/blog/receive-phone-call-python-flask-plivo/" target="_blank">create a simple Flask application to handle incoming calls on a Plivo number</a>. </p><p>Once you know how to use Python with Plivo to make and receive calls, you can move into specific use cases, such as <a href="https://www.plivo.com/docs/voice/use-cases/call-forwarding/python/#xml" target="_blank">call forwarding</a> and <a href="https://www.plivo.com/docs/voice/use-cases/ivr/python/#xml" target="_blank">interactive voice response</a> systems. </p><h2>More resources</h2><p>From there, the sky’s the limit.
