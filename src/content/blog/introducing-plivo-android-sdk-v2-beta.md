<p>Plivo’s <a href="https://www.plivo.com/docs/sdk/client/android/reference/">Android SDK</a> v2 is now available in public beta. Android SDK v2 helps you make and receive calls in your Android applications. This new version of the SDK is backward-compatible with the older version.</p><h3>What’s new in v2</h3><p>Android SDK v2 includes major enhancements in architecture support, security, and stability.</p><h3>Wider architecture support</h3><p>The new SDK is fully compatible with all major mobile architectures:</p><ul><li>arm64-v8a</li><li>armeabi</li><li>armeabi-v7a</li><li>mips</li><li>mips64</li><li>x86</li><li>x86_64</li></ul><h3>Enhanced security</h3><p>The new SDK uses an OpenSSL v1.0.2k encrypted connection for SIP signaling. This added layer of security eliminates the risk from man-in-the-middle attacks.</p><h3>Enhanced stability</h3><p>The new SDK promises improved performance and stability over older versions, thanks to numerous bug fixes and code optimizations made based on crash reports received from existing SDK users.</p><h3>Getting started</h3><p>We’ve created a few resources to help you get started building Android apps using the Plivo Android SDK v2. The <a href="https://www.plivo.com/docs/sdk/client/android/reference/">SDK Reference</a> page documents all the APIs that the SDK exposes. The step-by-step <a href="https://www.plivo.com/docs/sdk/client/android/overview/">getting started guide</a> should help get you live within minutes.</p><p>We’ve also developed a <a href="https://github.com/plivo/plivo-android-sdk2-examples">sample Android application</a> that implements all the features of the SDK. The example app also showcases how to implement a background Keep Alive service to receive incoming calls even when your app is not running.</p><p>Download the SDK <a href="https://s3-us-west-1.amazonaws.com/helpers.plivo.com/android/android-sdk2-beta.zip">directly</a> or from the <a href="https://www.plivo.com/docs/sdk/client/android/overview/">Android SDK V2 documentation</a> page.</p><h3>What’s coming next?</h3><p>The new Android SDK v2 is fully compatible to handle push notifications for incoming calls via Google Cloud Manager (GCM) and Firebase. We’re working on changes in our back-end stack to facilitate push notifications for incoming calls. Once these changes are rolled out, app users will be able to receive incoming calls even when the app’s background service isn’t running, thus saving bandwidth and battery usage.<br></p><p></p>
