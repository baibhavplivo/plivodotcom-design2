<p>This guide shows how to write an autoresponder for SMS text messages. Autoresponders can streamline marketing campaigns and subscription signups and reduce the amount of work humans have to do. You can create a text autoresponder either by using our PHLO visual workflow builder or our <a href="https://www.plivo.com/sms/">SMS API</a>. Follow the instructions in one of the tabs below.</p><p>You can create and deploy a PHLO to implement an SMS autoresponder with a few clicks on the PHLO canvas.</p><h2>How it works</h2><figure style="padding-bottom:33.75%" data-rt-max-width="" data-rt-max-height="33.75%" data-rt-dimensions="0:0" data-page-url=""><div><iframe src="https://www.youtube.com/embed/sxlJkKEkgHI?rel=0&amp;modestbranding=1" frameborder="0" allowfullscreen=""></iframe></div></figure><h2>Prerequisite</h2><p>To get started, you need a Plivo account — <a href="https://console.plivo.com/accounts/register/">sign up</a> with your work email address if you don’t have one already. If you’re using a Plivo Trial account, you can send SMS messages only to phone numbers that have been verified with Plivo. You can verify (sandbox) a number by going to the console’s Phone Numbers &gt; <a href="https://console.plivo.com/sandbox-numbers/">Sandbox Numbers</a> page.</p><h2>Create the PHLO</h2><p>To create a PHLO, visit the <a href="https://console.plivo.com/phlo/list/">PHLO</a> page of the Plivo console. If this is your first PHLO, the PHLO page will be empty.</p><ul><li>Click Create New PHLO.</li><li>In the Choose your use case pop-up, click Build my own. The PHLO canvas will appear with the Start node.<strong>Note:</strong> The Start node is the starting point of any PHLO. It lets you trigger a PHLO to start upon one of three actions: incoming SMS message, incoming call, or API request.</li><li>If you plan to use a dynamic payload — passing values through parameters when you trigger the PHLO from your application — click on the Start node to open the Configuration pane. Under API Request, enter key names for the variables you want to use in your payload — for example, a list of numbers to send an SMS message to.</li><li>Once you’ve configured the node, click Validate to save the configuration.</li><li>From the list of components on the left-hand side, drag and drop the Send Message component onto the canvas. When a component is placed on the canvas it becomes a node.</li><li>Draw a line to connect the Start node’s Incoming Message trigger state to the Send Message node.</li><li>In the Configuration pane at the right of the canvas, configure the Send Message node with a sender ID in the From field. Enter the destination number you wish to send a message to in the To field. Put your message in the Text field.<strong>Note:</strong> You can define a static payload by specifying values when you create the PHLO, or define a dynamic payload by passing values through <a href="https://shopify.github.io/liquid/">Liquid</a> templating parameters when you trigger the PHLO from your application.</li></ul><video class="d-pointer-events" width="100%" height="auto" autoplay="autoplay" controls="" loop="" disablepictureinpicture="" controlslist="nodownload">
 &nbsp; &nbsp;<source src="https://www.plivo.com/assets/posts/images/phlo/sms-autoresponder.mp4" type="video/mp4"></video><ul><li>Once you’ve configured the node, click Validate to save the configuration.</li><li>After you complete the configuration, give the PHLO a name by clicking in the upper left, then click Save.</li></ul><h2>Assign the PHLO to a Plivo number</h2><p>Once you’ve created and configured your PHLO, assign it to a Plivo number.</p><ul><li>On the <a href="https://console.plivo.com/active-phone-numbers/">Numbers</a> page of the console, under Your Numbers, click the phone number you want to use for the PHLO.</li><li>In the Number Configuration box, select PHLO from the Application Type drop-down.</li><li>From the PHLO Name drop-down, select the PHLO you want to use with the phone number, then click Update Number.</li></ul><figure><div><img src="https://uploads-ssl.webflow.com/656ac26f3a3f6e1cc08ba7a5/658016b2a1a7cc3479ef2cfe_assign-phlo-autoresponder.png" alt="How to Build a Text Autoresponder Using PHLO | Plivo" width="auto" height="auto" loading="auto"></div></figure><h2>Test</h2><p>You can now send a message to your Plivo phone number and see how the autoresponder works.</p><p>For more information about creating a PHLO application, see the <a href="https://www.plivo.com/docs/phlo/">PHLO Getting Started guide</a>. For information on components and their variables, see the <a href="https://www.plivo.com/docs/phlo/components/">PHLO Components Library</a>.<br></p>
