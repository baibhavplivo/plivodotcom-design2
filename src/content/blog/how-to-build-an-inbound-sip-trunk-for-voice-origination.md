<p>Plivo’s <a href="https://www.plivo.com/phlo/">PHLO</a> visual workflow builder lets you quickly create and configure a SIP trunk call forwarding system for your business. You can migrate your existing SIP trunk hosted voice application to PHLO, which then forwards your Plivo number’s incoming calls to an IP PBX and SIP endpoint.</p><h2>Configure your inbound SIP trunk</h2><p>Plivo provides <a href="https://www.plivo.com/sip-trunking/">SIP trunk</a> configuration instructions for the most popular SIP trunking platforms, including <a href="https://www.plivo.com/docs/sip-trunking/interconnection-guides/freeswitch/#configuring-an-inbound-trunk">Freeswitch</a>, <a href="https://www.plivo.com/docs/sip-trunking/interconnection-guides/asterisk-secure-trunking/">Asterisk</a>, <a href="https://www.plivo.com/docs/sip-trunking/interconnection-guides/freepbx/#configuring-an-inbound-trunk">FreePBX</a>, <a href="https://www.plivo.com/docs/sip-trunking/interconnection-guides/3cx/">3CX</a>, and <a href="https://www.plivo.com/docs/sip-trunking/interconnection-guides/fusionpbx/">FusionPBX</a>.</p><h2>Prerequisites</h2><p>To get started, you need a Plivo account — <a href="https://console.plivo.com/accounts/register/">sign up</a> with your work email address if you don’t have one already. To receive incoming calls, you must have a voice-enabled <a href="https://www.plivo.com/virtual-phone-numbers/">Plivo phone number</a>. You can rent numbers from the <a href="https://console.plivo.com/active-phone-numbers/">Numbers</a> page of the Plivo console, or by using the <a href="https://www.plivo.com/docs/numbers/">Numbers API</a>.</p><h2>Create a PHLO to forward inbound calls</h2><p>To forward an incoming call, you can create and deploy a PHLO with a few clicks on the PHLO canvas.</p><figure><div><img src="https://uploads-ssl.webflow.com/656ac26f3a3f6e1cc08ba7a5/658008ee80e95f1cddc3d750_easytrunk2phlo.gif" alt="Create a PHLO" width="auto" height="auto" loading="auto"></div></figure><ul><li>On the side navigation bar, click <strong>PHLO</strong>. The PHLO page will appear and display your existing PHLOs, if any exist. If this is your first PHLO, then the PHLO page will be empty.</li><li>Click <strong>Create New PHLO</strong> to build a new PHLO.</li><li>In the <strong>Choose your use case</strong> window, choose the <strong>Call Forwarding</strong> use case. The PHLO canvas will appear with a prebuilt template for the <a href="https://www.plivo.com/docs/phlo/use-case-guides/call-forwarding/">Call Forwarding</a> use case.</li><li>Configure the <strong>Call Forward</strong> node.</li></ul><ul>
 &nbsp; &nbsp; &nbsp;<li><strong>From</strong> field: This will be already configured with . It’s the caller ID to display to the recipient of the forwarded call.</li>
 &nbsp; &nbsp; &nbsp;<li>
 &nbsp; &nbsp; &nbsp; &nbsp;<p><strong>To</strong> field: List of recipients to forward the incoming call and to bridge into the active call. The list can contain IP PBXes and SIP endpoints separated by a comma:</p>

 &nbsp; &nbsp; &nbsp; &nbsp;<ul>
 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;<li>sip:1NPANXXYYYY@sip.example.com</li>
 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;<li>sip:phonenumber@192.0.0.1</li>
 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;<li>sip:sipendpoint@phone.plivo.com</li>
 &nbsp; &nbsp; &nbsp; &nbsp;</ul>
 &nbsp; &nbsp; &nbsp; &nbsp;<div class="notice-box-response">
<p><strong>Note:</strong> You must choose sequential for call forwarding configuration setting so that it will be dialed sequentially on failover.</p>
</div>
 &nbsp; &nbsp; &nbsp;</li>
 &nbsp; &nbsp;</ul><ul><li>Once you’ve configured the node, click <strong>Validate</strong> to save the configurations.</li><li>After you complete the configuration, provide a name for your PHLO and click <strong>Save</strong>.</li></ul><h2>Assign the PHLO to a Plivo number to forward incoming calls</h2><p>Once you’ve created and configured your PHLO, assign your PHLO to a Plivo number.</p><ol><li>Log in to the <a href="https://console.plivo.com/dashboard/">Plivo console</a>.</li><li>On the Product Navigation bar, click <strong>Phone Numbers</strong>.</li><li>On the Numbers page, under <strong>Your Numbers</strong>, click the phone number you want to use for the PHLO.</li><li>In the <strong>Number Configuration</strong> window, select <strong>PHLO</strong> from the <strong>Application Type</strong> list.</li><li>From the <strong>PHLO Name</strong> list, select the PHLO you want to use with the phone number, and then click <strong>Update Number</strong>.</li></ol><figure><div><img src="https://uploads-ssl.webflow.com/656ac26f3a3f6e1cc08ba7a5/658008ee54765f5aad66ad8c_assign-phlo-forward.png" alt="Assign a Phone Number" width="auto" height="auto" loading="auto"></div></figure><h2>Test</h2><p>You can now make a call to your Plivo phone number and see how the inbound call is forwarded to the list of IP PBXes and endpoint specified in the PHLO.</p><p>For more information about creating a PHLO app, see the <a href="https://www.plivo.com/docs/phlo/">PHLO User Guide</a>.<br>For information on components and their variables, see the <a href="https://www.plivo.com/docs/phlo/#components-library">PHLO Components Library</a>.</p><p>Haven’t tried <a href="https://www.plivo.com/">Plivo</a> yet? Getting started is easy and only takes minutes. <a href="https://console.plivo.com/accounts/register/">Sign up</a> today.<br></p>
