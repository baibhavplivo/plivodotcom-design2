<p><strong>EDIT:</strong> The Go SDK is now out of beta — see the <a href="https://www.plivo.com/blog/introducing-plivo-server-side-sdks-v4/">GA release blog post</a>.</p><p>We’re excited to announce the public beta of our new Plivo Go SDK 4.0.</p><p>You might wonder why the first version of the Go SDK is 4.0. It’s because we’re rewriting all of our SDKs from scratch and we wanted to give them all a common version number.</p><h2>Features</h2><ul><li>The Go SDK works with all versions of Go &gt;= 1.0.x. It has been tested against all minor versions from 1.0.x through 1.9.x.</li><li>The API interfaces are consistent and guessable.</li><li>The new SDK has excellent IDE support, including but not limited to Atom, Vim, and Sublime Text, without the need to juggle between documentation and code.</li></ul><h2>Documentation</h2><p>Read our <a href="https://www.plivo.com/docs/sdk/server/go-sdk/">getting started guide for the Go SDK</a>, and our guides for setting up a Go dev environment for <a href="https://www.plivo.com/docs/sdk/server/set-up-go-dev-environment-api-xml-voice/">voice calls</a>, <a href="https://www.plivo.com/docs/sdk/server/set-up-go-dev-environment-api-messaging/">messaging</a>, and <a href="https://www.plivo.com/docs/sdk/server/set-up-go-dev-environment-phlo/">PHLO</a>.</p><p></p>
