<p>Let’s get 2014 started on the right track by keeping in touch with friends and family throughout the year using this nifty Plivo and Mailgun API project. Connect your Plivo phone number to your email via your Mailgun account and receive email notifications every time you get an SMS message. You can even reply to messages straight from your email inbox.</p><p>You can build this gateway for free using:</p><ul><li>A <a href="https://console.plivo.com/accounts/register/">free Plivo trial account</a>. Once you’ve registered you can claim a free local US phone number by clicking <strong>Buy Number</strong> from the <a href="https://console.plivo.com/active-phone-numbers/">Phone Numbers page</a> of the Plivo console.</li></ul><ul><li>A <a href="https://signup.mailgun.com/new/signup">free Mailgun account</a>.</li></ul><ul><li>A <a href="https://signup.heroku.com/identity">free Heroku account</a>. If you’re new to Heroku, visit the <a href="https://devcenter.heroku.com/start">Getting Started</a> page for guides on setting up your account. Create a new app and install the free RedisToGo add-on, which provides functionality for Redis, a key-value NoSQL database. In our example, we call our app your-heroku-app.</li></ul><h2>Push the application code to Heroku</h2><p>Once you’ve created accounts on all three platforms, download the sample code for our <a href="https://github.com/plivo/apps/tree/master/plivomessenger">plivomessenger</a> application using Git. if you’re not familiar with Git, you can learn about it from <a href="https://git-scm.com/book/en/v2">Pro Git</a>, a free online ebook.</p><style>

<span class="nv">$ </span>git commit <span class="nt">-m</span> <span class="s2">"init"</span>

<span class="nv">$ </span>git push heroku master
<span class="nv">$ </span>heroku scale <span class="nv">web</span><span class="o">=</span>1</code></pre></figure><h2>Configure your credentials in Heroku</h2><p>This SMS-to-email gateway requires your Plivo authentication credentials and phone number, which you can find on the overview page of the Plivo <a href="https://console.plivo.com/dashboard/">console</a>. Check out Heroku’s <a href="https://devcenter.heroku.com/articles/config-vars">variable configuration tutorial</a> if you need help.</p><figure><pre><code class="language-bash" data-lang="bash"><span class="nv">$ </span>heroku config:set <span class="nv">PLIVO_AUTH_ID</span><span class="o">=</span>your-plivo-auth-id
<span class="nv">$ </span>heroku config:set <span class="nv">PLIVO_AUTH_TOKEN</span><span class="o">=</span>your-plivo-auth-token
<span class="nv">$ </span>heroku config:set <span class="nv">PLIVO_NUMBER</span><span class="o">=</span>your-SMS-enabled-plivo-number&lt;/code&gt;</code></pre></figure><p><br></p><p>Use any of your Plivo phone numbers from the <a href="https://console.plivo.com/active-phone-numbers/">Phone Numbers</a> page of the Plivo console.</p><p>You also need to supply your Mailgun credentials, which you can get from the <a href="https://mailgun.com/cp">Mailgun Control Panel</a>.</p><figure><pre><code class="language-bash" data-lang="bash"><span class="nv">$ </span>heroku config:set <span class="nv">MAILGUN_ACCESS_KEY</span><span class="o">=</span>your-mailgun-access-key
<span class="nv">$ </span>heroku config:set <span class="nv">MAILGUN_SERVER_NAME</span><span class="o">=</span>yourdomain.com</code></pre></figure><h2>Configure Plivo</h2><p>With Heroku properly configured, it’s time to let Plivo know where to find your SMS-to-email gateway application.</p><p>Visit the Voice &gt; <a href="https://console.plivo.com/voice/applications/">Applications</a> page of the Plivo console and click <strong>Add New Application</strong>. Give a name to your application — we called ours “SMS Forward.” Set the Primary Answer URL and Message URL to</p><p>https://your-heroku-app.herokuapp.com/forward/?Emails=youremail@yourdomain.com. Then click <strong>Create Application</strong>.</p><p>Now assign your Plivo phone number to the application. Go to the <a href="https://console.plivo.com/number">Phone Numbers</a> page and select the phone number you want to use. On the next screen, from the Application Type drop-down, select XML Application. From the Plivo Application drop-down, select the name of your application. Click <strong>Update Number</strong> to save your changes.</p><p>It’s not required, but it’s good practice to run the application locally to make sure everything is working properly. To do that, paste this link in your browser. Be sure to insert your own Heroku application URL and your email address:</p><p>https://your-heroku-app.herokuapp.com/forward/?Emails=youremail@yourdomain.com&amp;From=1000&amp;To=1000&amp;Text=Test</p><p>The browser should display an empty &lt;Response&gt; XML document.</p><h2>Configure your Plivo credentials in Mailgun</h2><p>Now, time to do a little configuration for Mailgun. First, configure your DNS with a subdomain for incoming email. For this example, we set up my.plivo.com as our email domain by adding DNS records on <a href="https://aws.amazon.com/route53/">Amazon Route 53</a>. Most domain registrars offer similar zone editing software.</p><figure style="max-width:1209px" data-rt-max-width="1209px"><div><img src="https://uploads-ssl.webflow.com/656ac26f3a3f6e1cc08ba7a5/65850cb2b056acf35980cc3f_Mailgun-Plivo-DNS.png" alt="Set up your DNS" width="auto" height="auto" loading="auto"></div></figure><h3>Let Mailgun know where to send your email</h3><p>Next, in your Mailgun account, set up a mail domain for my.yourdomain.com:</p><figure style="max-width:1211px" data-rt-max-width="1211px"><div><img src="https://uploads-ssl.webflow.com/656ac26f3a3f6e1cc08ba7a5/65850cb2ddadde3459ff43b3_Mailgun-Domain.png" alt="Setup your mail domain in Mailgun" width="auto" height="auto" loading="auto"></div></figure><p>Then add your route to Mailgun. Click on the Route tab in Mailgun and add a new route with the following parameters:</p><p>Filter Expression: match_recipient(“^(.*)@ my.yourdomain.com$”) <br>Actions: forward("http://your-heroku-app.herokuapp.com/reverse/?")</p><figure style="max-width:964px" data-rt-max-width="964px"><div><img src="https://uploads-ssl.webflow.com/656ac26f3a3f6e1cc08ba7a5/65850cb27cec5829d023b1df_Mailgun-Routes.png" alt="Add your Route to Mailgun" width="auto" height="auto" loading="auto"></div></figure><p>That’s it! You can now try out your SMS-to-email gateway.</p><h2>Test SMS to email</h2><p>Try sending a text message to your SMS-enabled Plivo number. It should arrive in the inbox at youremail@yourdomain.com, as configured. The email will have originated from yourphonenumber@my.yourdomain.com.</p><h2>Test email to SMS</h2><p>Now try replying to that email. The reply should go back to your phone. In our application, we only let email forward to numbers from which we received a request.</p>
