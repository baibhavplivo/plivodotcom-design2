<p><a href="https://www.plivo.com/">Plivo</a> has a lot of great capabilities built into its cloud communications platform. If you’re a Node.js developer, this post is for you: Learn how easy it is to send and receive SMS text messages and make and receive voice calls with Node, the Plivo Node.js SDK, and Plivo APIs.</p><h3>How to send and receive SMS messages</h3><p>If you’re interested in messaging applications, Plivo’s cloud communications platform makes things easy for you. <a href="https://www.plivo.com/blog/send-sms-in-node-express/" target="_blank">Sending an SMS message</a> requires just 10 lines of code; receiving messages takes just 22, or you can <a href="https://www.plivo.com/blog/receive-respond-sms-node-expressjs-plivo/" target="_blank">create a Node.js application to receive and respond to SMS messages</a>.</p><h3>How to send and receive MMS messages</h3><p><a href="https://www.plivo.com/blog/send-mms-in-node/" target="_blank">Sending MMS messages</a> is equally simple, and we’ve written a <a href="https://www.plivo.com/blog/how-to-receive-and-respond-incoming-mms-messages-in-node/" target="_blank">guide to receiving and responding to MMS messages with Node.js</a>.</p><h3>How to get set up and make and receive phone calls</h3><p>Start by reading <a href="https://www.plivo.com/blog/make-phone-calls-in-node-phlo/" target="_blank">how to make a phone call from a Node application using Plivo’s Voice API</a>. That post covers how to install Plivo’s Node.js SDK, which makes it easier to build applications that use Plivo. The SDK provides objects, functions, and methods to handle common tasks associated with building voice and messaging applications.</p><p>Once you have that in place, you can use <a href="https://www.plivo.com/phlo/" target="_blank">PHLO</a>, Plivo’s visual workflow design studio, to set up the workflow of the call, then trigger the PHLO with a few lines of Node code.</p><p>Once you know how to use Node with Plivo to make and receive calls, you can move into specific use cases, such as <a href="https://www.plivo.com/docs/voice/use-cases/call-forwarding/node/#xml" target="_blank">call forwarding</a> and <a href="https://www.plivo.com/docs/voice/use-cases/ivr/node/#xml" target="_blank">interactive voice response</a> systems. </p><h3>More resources</h3><p>From there, the sky’s the limit. We support you all the way with our <a href="https://www.plivo.com/docs/sdk/server/node-sdk/">Node.js SDK</a>, <a href="https://www.plivo.com/docs/sms/quickstart/node-expressjs/" target="_blank">Quickstart guides</a> that include using Express.js and NestJS, and more than a dozen <a href="https://www.plivo.com/docs/sms/use-cases/send-an-sms/node/" target="_blank">use case guides</a>, and if those aren’t enough, you can <a href="https://support.plivo.com/hc/en-us/requests/new?ticket_form_id=360000156292" target="_blank">contact our support team</a> for <a href="https://www.plivo.com/blog/cpaas-technology-lessons-support/" target="_blank">white glove support</a>.</p><h4>A journey of a thousand miles Plivo use cases begins with a single step: <a href="https://console.plivo.com/accounts/register/" target="_blank">sign up</a> for a free trial account today.<br></h4>
