<p>Plivo has a lot of great capabilities built into its cloud communications platform. If you’re a PHP developer, this post is for you: Learn how easy it is to send and receive SMS text messages and make and receive voice calls with PHP, the Plivo PHP SDK, and Plivo APIs.</p><h3>How to send and receive SMS messages</h3><p>If you’re interested in messaging applications, Plivo’s cloud communications platform makes things easy for you. <a href="https://www.plivo.com/blog/send-sms-in-php/" target="_blank">Sending an SMS message</a> requires just 10 lines of code; receiving messages takes only nine.</p><h3>How to send and receive MMS messages</h3><p><a href="https://www.plivo.com/blog/send-mms-in-php/" target="_blank">Sending MMS messages</a> is equally simple, and we’ve written a <a href="https://www.plivo.com/blog/how-to-receive-and-respond-incoming-mms-messages-in-php/" target="_blank">guide to receiving and responding to MMS messages with PHP</a>.</p><h3>How to get set up and make and receive phone calls</h3><p>Start by reading <a href="https://www.plivo.com/blog/make-phone-calls-in-php/" target="_blank">how to make a phone call from a PHP application using Plivo’s Voice API</a>. That post covers how to install Plivo’s PHP SDK, which makes it easier to build applications that use Plivo. The SDK provides objects, functions, and methods to handle common tasks associated with building voice and messaging applications.</p><p>Once you have that in place, you can use <a href="https://www.plivo.com/phlo/" target="_blank">PHLO</a>, Plivo’s visual workflow design studio, to set up the workflow of the call, then trigger the PHLO with a few lines of PHP code.</p><p>Once you know how to use PHP with Plivo to make and receive calls, you can move into specific use cases, such as <a href="https://www.plivo.com/docs/voice/use-cases/call-forwarding/php/#xml" target="_blank">call forwarding</a> and <a href="https://www.plivo.com/docs/voice/use-cases/ivr/php/#xml" target="_blank">interactive voice response</a> systems. </p><h3>More resources</h3><p>From there, the sky’s the limit.
