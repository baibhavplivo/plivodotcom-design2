<p>You can now send and receive MMS messages in the United States and Canada on all MMS-enabled Plivo long code phone numbers. With <a href="https://www.plivo.com/mms/">Plivo MMS</a>, businesses now have a new channel to use to communicate with their customers, and the ability to create engaging visual experiences with pictures, audio, and video.</p><p>Our early MMS customers are using the new feature in several ways:</p><ul><li><strong>Marketing campaigns:</strong> To improve open rates for marketing campaigns using rich media such as pictures and videos.</li><li><strong>Last-mile logistics:</strong> To identify personal information and shipping details using pictures and videos shared during delivery.</li><li><strong>Collecting account information:</strong> To collect account-related information such as pictures and receipts using MMS.</li></ul><h2>Sending and receiving MMS messages</h2><p>Sending and receiving MMS messages is like sending and receiving regular text messages on the Plivo platform. You can make a POST request to the message resource URL or an API request to specify the image link in the Media URL parameter.</p><figure><div><img src="/images/blog/65828aa188c080e6f49078a0_MMS_image.jpeg" alt="Sending and Recieving MMS Messages" width="auto" height="auto" loading="auto"></div></figure><p>For more information, check out our <a href="https://www.plivo.com/docs/sms/api/message/#send-a-message">API reference documentation</a>.</p><h2>Our MMS API includes many powerful features:</h2><ul><li><strong>Multiple sender types</strong><br>Send and receive MMS using any number type (long code, toll-free, short code) across the US and Canada.</li><li><strong>Variety of formats</strong><br>Support for pictures (JPEGs and GIFs), emojis, audio (MP3), and video (MP4) files.</li><li><strong>Built-in multimedia storage</strong><br>Plivo provides built-in multimedia storage for both inbound and outbound MMS messaging so you can access and reuse media.</li><li><strong>Intelligent picture resizing (coming soon)</strong><br>The MMS API intelligently resizes images to meet carrier guidelines and to ensure delivery across different carrier networks.</li><li><strong>Automatic MMS converter (coming soon)</strong><br>Automatic conversion of multimedia messages into SMS messages when sending messages to MMS-incompatible phone numbers to ensure better deliverability.</li></ul><p>With our server SDKs and <a href="https://www.plivo.com/docs/sms/use-cases/send-an-mms/node/">ready-to-use code samples</a>, it’s simple to incorporate sending MMS message in your applications.</p><p>MMS is available for the US and Canada on long code phone numbers, and US short codes and toll-free numbers are both on the roadmap. Pay-as-you-go pricing for the US starts at $0.016 to send and $0.008 to receive, with volume discounts on committed spend as you scale your usage.<br></p><p></p>
