<p><strong>EDIT:</strong> The Ruby SDK is now out of beta —see the <a href="https://www.plivo.com/blog/introducing-plivo-server-side-sdks-v4/">GA release blog post</a>.</p><p>We’re excited to announce the public beta of the Plivo Ruby SDK 4.0, which offers more features and improved performance. New features will be added to this version going forward. Ruby helper library 0.3.x, now called the legacy version, will receive only critical bug fixes and security patches.</p><p>You might wonder why we changed our version number from 0.3.9 to 4.0. It’s because we’re rewriting all of our SDKs from scratch and we wanted to give them all a common version number.</p><h3>What’s new</h3><ul><li>The new SDK works with Ruby &gt;= 2. It has been tested against 2.0.0, 2.1, 2.2.0, 2.3.0, 2.4.0, and 2.5-dev.</li><li>The API interfaces are consistent and guessable.</li><li>The SDK handles pagination automatically when listing all objects of a resource.</li><li>The SDK has excellent IDE support, including but not limited to PyCharm, Atom, Vim, and Sublime Text, without the need to juggle between documentation and code.</li></ul><h3>Documentation</h3><p>Read our <a href="https://www.plivo.com/docs/sdk/server/ruby-sdk/">getting started guide for the Ruby SDK</a>, and our guides for setting up a Ruby dev environment for <a href="https://www.plivo.com/docs/sdk/server/set-up-ruby-dev-environment-api-xml-voice/">voice calls</a>, <a href="https://www.plivo.com/docs/sdk/server/set-up-ruby-dev-environment-api-messaging/">messaging</a>, and <a href="https://www.plivo.com/docs/sdk/server/set-up-ruby-dev-environment-phlo/">PHLO</a>.</p><p></p>
