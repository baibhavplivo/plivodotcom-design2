<h2>How to evaluate an SMS API provider</h2><p>Virtually every business can enhance their relationships with their customers by taking advantage of SMS text messaging to stay in touch. Consider some possible use cases:</p><ul><li>Order status updates</li><li>Delivery and shipping notifications</li><li>Financial transaction alerts</li><li>Appointment reminders</li><li>Two-factor authentication and password resets</li></ul><p>What about your business? Bet you’ve already thought of a way to use text messaging, and you want to consider how to start a pilot project — or maybe you already use an SMS API, but you want to add a second one for fault tolerance. Now you need to decide what SMS API provider to use.</p><p>The simple answer is: use Plivo. But if you’re one of those troublesome people who won’t settle for simple answers and require evidence before making a decision, we’ve put together a <a href="https://www.plivo.com/resources/how-to-choose-an-sms-api-provider-the-complete-guide/" target="_blank">guide to choosing an SMS API provider</a>. It walks you through the evaluation process, from figuring out how you want to use text messaging to highlighting the features you should look for. This post gives you a taste of the contents, but you should download the guide for the full story.</p><h3>What to look for</h3><p>Focus your evaluation on a few broad factors about each platform: features, geographic reach, and pricing.</p><p>By features, we’re talking about support for Multimedia Messaging Service (MMS), long code numbers (including <a href="https://www.plivo.com/blog/10dlc-10-digit-long-code/">10DLC</a>), short codes, and toll-free numbers, as well as things like <a href="https://www.plivo.com/blog/plivo-rate-limiting-and-message/">rate limiting and message queuing</a>, <a href="https://www.plivo.com/docs/sms/concepts/encoding-and-concatenation/#encoding-types">support for GSM and Unicode characters</a> and <a href="https://support.plivo.com/hc/en-us/articles/360048079132">smart encoding</a>, and <a href="https://www.plivo.com/docs/sms/concepts/encoding-and-concatenation/#concatenation">long message concatenation</a>.</p><p>Geographic reach is an obvious factor. Your platform should be able to send messages to every country and region where you might have customers — and where you might have them in the future. Just to be on the safe side, look for a partner with a <a href="https://support.plivo.com/hc/en-us/articles/360048079132">global presence</a>, including relationships with in-country Tier 1 carriers with direct interconnectivity to local telecom network operators.</p><p>And while pricing isn’t necessarily your primary purchase criterion, it’s something everyone has to deal with. Look for transparent pricing, published on the vendor’s website. If you compare other platforms’ rates with <a href="https://www.plivo.com/sms/pricing/us/">Plivo’s prices</a>, you’ll find we’re generally lower across the board.</p><h3>The best SMS API</h3><p>Features, reach, and pricing are important factors to consider.
