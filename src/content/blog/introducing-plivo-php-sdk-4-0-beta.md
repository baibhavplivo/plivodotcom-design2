<p><strong>EDIT:</strong> The PHP SDK is now out of beta — see the <a href="https://www.plivo.com/blog/introducing-plivo-server-side-sdks-v4/">GA release blog post</a>.</p><p>We’re excited to announce the public beta of the Plivo PHP SDK 4.0, which offers more features and improved performance. New features will be added to this version going forward. PHP helper library 1.1.x, now called the legacy version, will receive only critical bug fixes and security patches.</p><p>You might wonder why we changed our version number from 1.1.7 to 4.0. It’s because we’re rewriting all of our SDKs from scratch and we wanted to give them all a common version number.</p><h3>What’s new</h3><ul><li>The new SDK works with PHP 5.5+ and 7.0+. It has been tested against the versions 5.5, 5.6, 7.0, 7.1, and the nightly version</li><li>The API interfaces are consistent and guessable.</li><li>The new SDK has excellent IDE support, including but not limited to Atom, Vim, and Sublime Text, without the need to juggle between documentation and code</li></ul><h3>Documentation</h3><p>Read our <a href="https://www.plivo.com/docs/sdk/server/php-sdk/">getting started guide for the PHP SDK</a>, and our guides for setting up a PHP dev environment for <a href="https://www.plivo.com/docs/sdk/server/set-up-php-dev-environment-api-xml-voice/">voice calls</a>, <a href="https://www.plivo.com/docs/sdk/server/set-up-php-dev-environment-api-messaging/">messaging</a>, and <a href="https://www.plivo.com/docs/sdk/server/set-up-php-dev-environment-phlo/">PHLO</a>.<br></p>
