<p>Zentrunk, Plivo’s SIP trunking service, offers fast setup, instant number provisioning, exceptional voice quality, and pay-as-you-go pricing. Users manage Zentrunk from the same Plivo console they use for our <a href="https://www.plivo.com/sms/">SMS API</a> and <a href="https://www.plivo.com/voice/">Voice API</a> — but we’re now adding new ways of managing <a href="https://www.plivo.com/sip-trunking/">Zentrunk</a> functions programmatically to help customers streamline their workflows.</p><p>Until now, Zentrunk customers would log in to the Plivo console to map phone numbers to inbound trunks. Managing a large volume of numbers and trunks could become cumbersome and time-consuming.</p><p>To address this issue, Plivo has enhanced our Number API to allow mapping numbers to SIP trunks. You can now use the <a href="https://www.plivo.com/docs/numbers/api/account-phone-number#the-accountphonenumber-object">AccountPhoneNumber object</a> and pass an inbound trunk ID as an application ID to map a number to an inbound trunk. For all of the technical details, see our <a href="https://www.plivo.com/docs/numbers/api/account-phone-number#update-a-number">API reference documentation</a>.</p><p>Using the API lets Zentrunk customers automate mapping of numbers to inbound trunks programmatically, allowing businesses to manage large volumes of numbers and trunks efficiently. Using the developer-friendly API saves time and effort when mapping multiple numbers to trunks and minimizes the chance of human error. The mapping process in Zentrunk is now consistent with the same process in our SMS and Voice APIs, making for a smooth and unified experience across the platform.</p><p>The new mapping capabilities join our other <a href="https://www.plivo.com/blog/introducing-new-zentrunk-apis/">recently announced Zentrunk APIs</a>: the Trunk Properties API, which lets businesses create, modify, and update their SIP trunk configurations programmatically, and the Call Object API, which lets users retrieve Zentrunk call details records (CDR).</p><p>All of the new Zentrunk APIs are designed to enhance the user experience for Zentrunk administrators. And it must be working — global media company TMC named Zentrunk a 2023<a href="https://www.plivo.com/blog/award-tmc-2023/"> Communications Solutions Product of the Year</a>. To see how well SIP trunking can work for your organization, <a href="https://console.plivo.com/accounts/register/">sign up for a free trial</a> today.<br></p>
