<p><strong>EDIT:</strong> The Java SDK is now out of beta — see the <a href="https://www.plivo.com/blog/introducing-plivo-server-side-sdks-v4/">GA release blog post</a>.</p><p>We’re excited to announce the public beta of the Plivo Java SDK 4.0.</p><h2>Features</h2><ul><li>The new SDK works with Java 1.8+ (JDK 8+). It has been tested against Java 1.8 and Java 1.9.</li><li>The API interfaces are consistent and guessable.</li><li>The new SDK has excellent IDE support, including but not limited to Atom, Vim, and Sublime Text, without the need to juggle between documentation and code.</li></ul><h2>Documentation</h2><p>Read our <a href="https://www.plivo.com/docs/sdk/server/java-sdk/">getting started guide for the Java SDK</a>, and our guides for setting up a Java dev environment for <a href="https://www.plivo.com/docs/sdk/server/set-up-java-dev-environment-api-xml-voice/">voice calls</a>, <a href="https://www.plivo.com/docs/sdk/server/set-up-java-dev-environment-api-messaging/">messaging</a>, and <a href="https://www.plivo.com/docs/sdk/server/set-up-java-dev-environment-phlo/">PHLO</a>.</p><p></p>
